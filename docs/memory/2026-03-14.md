## Feishu 配置优化（从飞书触发）
用户反馈需要"更稳的配置，减少流式问题的概率"，已优化 OpenClaw 的 Feishu 集成配置：

### 修改内容
**文件**: `/home/ubuntu/.openclaw/openclaw.json`

**关键变更**:
- 将 `channels.feishu.blockStreaming` 从 `false` 改为 `true`
- 保持 `channels.feishu.streaming` 为 `false`

**修改前**:
```json
"streaming": false,
"blockStreaming": false,
```

**修改后**:
```json
"streaming": false,
"blockStreaming": true,
```

### 配置说明
- `streaming: false` - 明确禁用流式传输模式
- `blockStreaming: true` - 强制阻止任何尝试启用流式传输的行为，提供更稳定的保护

### 预期效果
- 减少因流式传输导致的消息丢失或乱序问题
- 提供更稳定的消息传输体验
- 避免因网络波动导致的流式连接中断

### 后续建议
- 如果 Gateway 需要重启以使配置生效，可以运行 `openclaw gateway restart`
- 观察后续使用中是否还有流式相关问题
- 如果需要恢复流式功能，可将 `blockStreaming` 改回 `false`
