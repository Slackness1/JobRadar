## 遇到的问题和解决

### 问题 1: 合并脚本中的 cursor 未定义
**错误**: `name 'cursor' is not defined`
**原因**: 在函数内部使用了外部定义的 cursor
**解决**: 将 cursor 定义移到函数内部，并正确传递 conn 参数

### 问题 2: 日期解析
**问题**: 日期格式多样，需要统一解析
**解决**: 实现多格式日期解析函数，支持 ISO 格式和中文日期

## 经验总结

1. **数据合并是高风险操作**
   - 务必先备份原始数据
   - 分步执行，每步验证
   - 保留原始文件不删除

2. **字段映射需要仔细设计**
   - 理解两个系统的字段含义
   - 统一命名规范
   - 处理空值和默认值

3. **去重策略至关重要**
   - 选择合适的去重键
   - 考虑时间戳优先级
   - 验证去重效果

4. **文档记录很重要**
   - 记录执行过程
   - 记录问题和解决方法
   - 生成详细报告
